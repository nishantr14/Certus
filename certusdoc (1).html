System Architecture

Four Stages.One Verdict.

Stage 01
Ingestion
Accept PDF or image. Poppler renders PDFs at 300 DPI. Tesseract v5.4 extracts text with bounding boxes. Full EXIF + file metadata captured.
PopplerTesseract 5.4300 DPIEXIF
Stage 02
Detection
Three agents run in parallel: Visual (ELA + MantraNet), Text (OCR forensics), Metadata (EXIF + QR + checksum). Each returns a score and reliability weight.
ParallelMantraNetELA
Stage 03
Fusion
Evidence-based fusion engine applies hard caps, convergence penalties, and cross-agent trust rules. DIS is reliability-weighted, not a simple average.
DISEvidence RulesTrust Weights

Output
Forensic PDF report with ELA heatmaps, per-agent breakdown, and verdict chain. REST API for pipeline integration.

Fusion Engine

DocumentIntegrity Score.

Weighted Fusion Formula
DIS = Σ(Rᵢ × Sᵢ) / ΣRᵢ
Rᵢ = Dynamic reliability weight per agent
Sᵢ = Agent integrity score (0 to 1)
R₁ = f(image resolution, BRISQUE score)
R₂ = f(OCR confidence, text coverage %)
R₃ = f(metadata field completeness)
